# Weather app
This app using The [Weather API](https://www.weatherapi.com/my/) and written in Kotlin with Android Studio.
***
## Rick and Morty app staсk:
* Using Retrofit for fetching data from Rick and Morty API;
* MVVM pattern;
* Coroutines;
* Navigation-fragment.
***
